A fully accessible multi-page portfolio website built with semantic HTML5 and WCAG accessibility standards.

- Semantic HTML5 structure
- Accessible navigation
- Skip-to-content links
- Keyboard navigable
- ARIA attributes
- SEO optimized
- Responsive design
